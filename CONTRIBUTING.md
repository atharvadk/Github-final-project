## How Can I Contribute?

All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

### Reporting Bugs
* **Check the Issue Tracker:** Before opening a new issue, please check if the bug has already been reported.
* **Provide Details:** Use a clear and descriptive title. Include steps to reproduce the bug, the expected behavior, and what actually happened.

### Suggesting Enhancements
* **Be Specific:** Explain why the enhancement would be useful and how it should work.
* **Scope:** Ensure the suggestion aligns with the goal of a Simple Interest Calculator.

### Pull Requests
1.  **Fork the repository** and create your branch from `main`.
2.  **Ensure your code follows the project style** (if applicable).
3.  **Include documentation** if you are adding a new feature.
4.  **Issue a Pull Request** with a detailed description of your changes.
